Le but du projet est de pratiquer les notions acquises dans les deux premières 
parties du cours sur Git et Github sur Openclassrooms.

Ce README.txt contiendra les une présentations du projet avec les modifications effectuées.

Dans ce projet, nous avons développé un projet java initialement avec affichage " Hello World".
Nous avons fait un premier commit suite à la création du projet et du fichier README.txt.
Il faut noter que nous avons aussi crée un fichier .gitignore qui contient les fichier à ignorer 
du projet lors des commit respectifs.

Nous avons ensuite modifié le " Hello World " par " Bonjour à tous les étudiants de Openclassrooms".
Suite à cette modifications, nous avons effectué un second commit. 

Nous allons ensuite envoyer notre repository sur GitHub et recupérer l'URL sous la forme 
https://github.com/enyo10/HelloWorld.
